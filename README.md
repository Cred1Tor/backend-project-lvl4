# Task Manager

System for managing tasks. Lets you create tasks, assign executors, change task statuses and tag them with labels. Requires registration and authorization.

[Heroku](https://fierce-badlands-18912.herokuapp.com/)

## Setup

```sh
$ make setup
```

## Run

```sh
$ make start
# Open localhost:5000
```
